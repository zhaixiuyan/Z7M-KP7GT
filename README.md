## 本项目为神舟Z7M-KP7GT的OC-EFI引导文件

基于**黑果小兵**的EFI进行完善，目前在11.6（Big Sur）上除了偶尔睡眠卡死外基本正常运行。

系统镜像可前往[**黑果小兵的部落**](https://blog.daliansky.net/)下载

### **已支持：**

- 显卡2048
- Wi-Fi（无需借助第三方客户端）
- USB3.0
- 蓝牙
- 触摸板

### **待验证：**

- 隔空投送（非苹果手机,未验证）

### **待完善：**

- 有点卡顿
